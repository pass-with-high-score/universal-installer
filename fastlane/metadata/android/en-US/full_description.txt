Universal Installer is a powerful and modern Android package installer that supports all APK formats.

🚀 Multi-format Support
Install .apk, .apks, .xapk, and .apkm files seamlessly. Universal Installer handles split APK bundles with ease, powered by the Ackpine library for maximum reliability.

🔍 APK Analysis
Before installing, preview detailed information about the APK:
• App name, icon, and version
• Package name and file size
• Required permissions
• Supported CPU architectures (ABIs)
• Supported languages
• Minimum Android version required

⚡ Shizuku Silent Install
Skip the system confirmation dialog with Shizuku integration. Install and uninstall apps silently — perfect for power users and developers.

🛡️ VirusTotal Security Scanning
Automatically scan APK files for malware before installation using the VirusTotal API. Stay safe with real-time threat detection from 70+ antivirus engines.

📱 App Manager
Browse, search, and uninstall installed apps. Filter between user and system apps with a clean, searchable interface.

🎨 Material You Design
Beautiful Material 3 UI with dynamic theming, supporting both light and dark modes. Smooth animations and modern design language throughout.

📂 File Association
Open APK files directly from any file manager — Universal Installer registers as a handler for all supported formats.

• Split APK support (APKS, XAPK, APKM)
